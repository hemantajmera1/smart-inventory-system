# Smart Inventory Management System

A web-based Smart Inventory Management System developed using PHP and MySQL.

## Features
- Product Management
- GST Billing System
- Invoice Generation
- Stock In / Stock Out
- User Authentication
- Admin Dashboard
- Supplier Management

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Setup
1. Download project
2. Move folder to htdocs
3. Import database in phpMyAdmin
4. Start Apache & MySQL
5. Run project in browser
